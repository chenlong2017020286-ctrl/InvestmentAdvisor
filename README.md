# InvestmentAdvisor

iOS股票行情监控应用 - 投资顾问

## 功能特性

- 📈 **股票行情监控** - 实时显示股票价格、涨跌幅、成交量
- 💰 **基金行情监控** - 显示基金净值、估算净值、涨跌幅  
- 🎯 **模拟盘交易** - 支持买入/卖出股票，实时持仓管理

## 技术栈

- Swift 5
- UIKit
- Xcode

## 安装

1. 克隆仓库
2. 使用Xcode打开 `InvestmentAdvisor.xcodeproj`
3. 选择模拟器或真机运行
